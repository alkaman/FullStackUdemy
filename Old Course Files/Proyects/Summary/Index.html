comparando las subfamilias, notamos que la diferencia entre las partes más finas y gruesas de las letras suele ser más exagerada mientras más moderna es la tipografía y suele ser más sutil mientras más viejo es el estilo de letra

Font moods

  1. Serif: Traditional, Stable, Respectable, Authorative. Typo Examples: Minion Pro, Trajan, Baskerville. You can see them on
  2. Sans-Serif: Sensible, Simple, Straightforward, more aprochable. Typo Examples: Helvetica, Avenir, Din.
  3. Script Personal, Creative, Elegant. Examples: Typo Freestyle Scrypt, Adios Script Pro y Snell Roundhand.
  4. Display: Friendly, Loud, Amusing. Typo Examples: Vag Rounded, Gin, Thirsty Rough.
  5. Modern: Stylish, Chic, Smart. Typo Examples: Sackers Gothic, Gotham, Futura.

II Sans-Serif

90° angle typo.

Dentro de la sub familia de la Sans-Serif, se suelen usar las typos que son Sensitive, simple, straightfoward, more friendly.

Generalmente para parrafos se usan Humanist typeface como Gill Sans, Tahoma, Verdana se usan para los parrafos (high readable and higly legible)


Examples:

  1. News Gothic: Grotesque Sans Serif Typo.
  2. Helvetica: Neo-Grotesque Sans Serif Typo.
  3. Gill Sans: Humanist Sans Serif Typo.
  4. Futura: Geometric Sans Serif Typo.

Things we look on a typo to be readable:

  1. Open Shapes Typo: The open part of the shape of the letter to be wider than others.
  2. Ample Intercharacter Spacing: Idem space between letters.
  3. Unambiguous forms: Que cada character se entienda cual es, bien diferenciado el 9 de la g, etc.
  4. Varying Proportions: Que una O mayuscula o un cero sean diferenciables.

Es recomendable usar solo dos fuentes por diseño en vez de multiples fuentes.

Combinación de typos

Para seleccionar las dos typos a usar para titulo/s y body nos basamos en:

  1. Para generar Similitud I:

    Usar typos con el mismo mood.

  2. Para generar Similitud II:

    De la misma epoca, time era, similares.

  3. Para generar contraste I:

    serif-ness??? Usar Serif y Sans Serif para titulo y body o viceverza, para crear más contraste, interacción y llamar la atención.

  4. Para generar contraste II:

    Weights: Haciendo diferencia por grosores, negrita, etc para el titulo y algo más fino para el body para generar contraste entre ambos.

Typos Prohibidos: Comic Sans, Papyrus, Viner, Curlz, kristen, Alfabeto Griego. Por que son muy dificiles de combinar y hacer que queden bien.
